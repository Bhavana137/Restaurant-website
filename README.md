# 🍽️ Tasty Bites Restaurant Website

A modern and responsive Restaurant Website developed using HTML and CSS. This project showcases a professional restaurant website with multiple pages, attractive design, food gallery, menu, offers, reservations, and contact information.

## 📌 Features

- Responsive Navigation Bar
- Home Page with Hero Section
- About Us Page
- Menu Page
- Food Gallery
- Special Offers Page
- Reservation Form
- Contact Page
- Attractive Background Images
- Hover Effects and Animations
- Mobile-Friendly Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
